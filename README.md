# Bulletin Board

## How to Use

Initialize database:

```
> cd BulletinBoards
> sqlite3 .db
sqlite> .read scripts/init_sqlite.sql
sqlite> .exit
```

Run the appication:

```
> cd BulletinBoard
> dotnet run
```
